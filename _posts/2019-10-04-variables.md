## Types of variable:
* string (string)  
* integer (int)  
* decimal (double)  
* boolean (bool)  
* character (char)  

When using variables in C#, you must define it before it can be used:
```csharp
string vString
int vInteger
double vDecimal
bool vBoolean
char vCharacter
```
